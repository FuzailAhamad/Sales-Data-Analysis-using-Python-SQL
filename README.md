# Sales Data Analysis using Python & SQL

## Project Overview
This project analyzes structured sales data to extract meaningful business insights using Python and SQL. It demonstrates real-world data handling, analysis, and visualization techniques suitable for entry-level Python and data roles.

## Tech Stack
- Python
- Pandas, NumPy
- Matplotlib
- SQLite

## Project Structure
sales-data-analysis/
├── data/
│   └── sales.csv
├── src/
│   ├── main.py
│   ├── analysis.py
│   └── database.py
├── output/
│   └── monthly_sales.png
├── README.md
└── requirements.txt

## Features
- Data cleaning and preprocessing using Pandas
- Revenue and product-wise sales analysis
- Monthly sales trend visualization
- Region-wise revenue analysis using SQL
- Modular and maintainable project structure

## Insights Generated
- Total revenue calculation
- Top-performing products identification
- Monthly sales trend analysis
- Region-wise revenue contribution

## How to Run

1. Install dependencies
pip install -r requirements.txt

2. Run the project
python src/main.py

3. Output
- Console displays sales insights
- Sales trend chart saved in output/monthly_sales.png
